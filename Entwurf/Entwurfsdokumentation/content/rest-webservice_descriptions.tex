\section{REST-Webservice-Spezifikation}
\subsection{Notation}
Atomare Werte werden mittels \declatomlbl{Name}\jsonatom{Name} angegeben, wobei \textit{Name} ein eindeutiger Bezeichner für den Wert ist.\\
JSON-Datenklassen werden mittels \declobjlbl{Name}\jsonobj{Name} angegeben, wobei \textit{Name} ein eindeutiger Bezeichner für die JSON-Datenklasse ist.\\
JSON-Objekte, die nur eine Teilmenge aller Attribute einer JSON-Datenklasse \jsonobj{Name} enthalten, werden mittels \jsonpart{Name}{$\textnormal{prop}_1, ..., \textnormal{prop}_n$} angegeben, wobei $\textnormal{prop}_1, ..., \textnormal{prop}_n$ die enthaltenen Attribute benennen.
